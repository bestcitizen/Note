## Layout : 구성 요소를 배치하기

- layout이란 특정 공간에 여러 구성 요소를 보기 좋게 효과적으로 배치하는 작업을 의미합니다.
- HTML에서는 `div` tag나 HTML5의 layout tag로 layout을 작성합니다.



## Layout에 사용하는 Tag

| Tag | 설명 |
| - | - |
| header |   |
| nav |  |
| main |  |
| section |  |
| article |  |
| aside |  |
| footer |  |
| div |  |
