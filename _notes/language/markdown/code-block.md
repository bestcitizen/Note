- code block은 2가지 방법으로 작성할 수 있습니다.




---




## 방법 1. `<pre><code>`


### 문법

```
<pre>
<code>
main( ) {
    puts("Hello, world!");
    return 0;
}
</code>
</pre>
```


### 결과

<pre>
<code>
main( ) {
    puts("Hello, world!");
    return 0;
}
</code>
</pre>

## 방법 2. \`\`\` (code block code)


### 문법

<pre>
<code>
```
main( ) {
    puts("Hello, world!");
    return 0;
}
```
</code>
</pre>


### 결과

```
main( ) {
    puts("Hello, world!");
    return 0;
}
```
