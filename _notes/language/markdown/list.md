## 순서 있는 목록

- 순서있는 목록은 숫자와 점을 사용합니다.

```
1. 하나
2. 둘
3. 셋
```

1. 하나
2. 둘
3. 셋




## 순서 없는 목록

- `*`, `+`, `-` 기호를 사용합니다.
    - 혼합하여 사용해도 하나만 사용했을 때와 결과가 같습니다.

```
* 하나 (*)
* 둘 (*)
* 셋 (*)

+ 하나 (+)
+ 둘 (+)
+ 셋 (+)

- 하나 (-)
- 둘 (-)
- 셋 (-)

* 기호 혼합 하나
+ 기호 혼합 둘
- 기호 혼합 셋

- tree
    - branch
        - leaf
        - branch
            - leaf
```

* 하나 (*)
* 둘 (*)
* 셋 (*)

+ 하나 (+)
+ 둘 (+)
+ 셋 (+)

- 하나 (-)
- 둘 (-)
- 셋 (-)

* 기호 혼합 하나
+ 기호 혼합 둘
- 기호 혼합 셋

- tree
    - branch
        - leaf
        - branch
            - leaf
