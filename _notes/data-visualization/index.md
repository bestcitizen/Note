- Data 시각화(data visualization)는 data 분석 결과를 쉽게 이해할 수 있도록 시각적으로 표현하고 전달되는 과정을 말합니다.
- Data 시각화의 목적은 도표(graph)라는 수단을 통해 정보를 명확하고 효과적으로 전달하는 것입니다.
