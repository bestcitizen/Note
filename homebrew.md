# HomeBrew

- brew list : homebrew로 설치된 application 조회
- brew install maven@3.8.1 : 특정 version install
- brew services : services 조회
- brew services start
- brew services stop
- brew services restart
- brew cleanup : install, update를 하며 누적된 package installer를 삭제함
